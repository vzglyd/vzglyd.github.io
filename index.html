      WebGPU not available
      

        This preview requires WebGPU. Try Chrome 113+, Edge 113+, or Safari 18+.
      

      

        On Linux (Chrome / Edge) enable the flag and relaunch:

        edge://flags/#enable-unsafe-webgpu  →  Enabled
      

      

        On Windows / macOS make sure hardware acceleration is on:

        edge://settings/system  →  Use hardware acceleration when available
      

    


    

      Having issues? Check browser GPU support →
    


    

      Serving over HTTP required
      

        Edge and Chrome block WebGPU on file:// pages. Serve this folder
        over a local HTTP server instead — pick any one-liner below, then open
        http://localhost:8080:
      

      

        

          python3 -m http.server 8080 --directory web-preview
          
        

        

          npx serve web-preview -l 8080
          
        

        

          cargo install basic-http-server && basic-http-server web-preview -a 127.0.0.1:8080
          
        

      

      
Run the command from the lume repository root.
